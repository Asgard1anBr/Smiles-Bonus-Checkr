# ✈️ Smiles Bonus Tracker

Monitor automático de bônus e promoções Smiles.

## 📱 Como instalar no celular

1. Acesse o link do GitHub Pages
2. **Android (Chrome):** 3 pontinhos > "Adicionar à tela inicial"
3. **iPhone (Safari):** Compartilhar > "Adicionar à Tela de Início"
